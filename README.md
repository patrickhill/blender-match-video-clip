A small Blender add-on that quickly configures your scene to match the selected strip in the Video Sequence Editor.

When run, it automatically:
	•	Sets the scene frame rate to match the strip
	•	Sets the scene frame range to the strip’s duration
	•	Sets the scene resolution to the strip’s source resolution
	•	Resets the strip’s transform scale to 1

This is useful when starting a new edit or importing footage and wanting the scene settings to immediately match the clip you’re working with.
